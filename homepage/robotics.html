Robotics Club

Peak Leveled Education Robotics Club builds robots to participate in the First Tech Challenge every year

In order to be as efficient as possible, we divide club members into teams to work on different elements of the robot

Teams:

They handle the portion of the robot that moves around the arena.

They handle the arm of the robot that guides the claw in order to reposition game elements.

They handle the portion of the robot that grabs game elements and places them to gain points

A very specific team for this year's competition, they handle the airplane launching mechanism of the robot to gain additional points

They link all of the other team's components into a controlable robot using Javascript
